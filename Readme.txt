Instruções para a execução do projeto

1 - O Docker engine deve estar instalado na máquina
2 - Entrar na pasta "Docker" a partir da raiz do projeto
3 - Executar o comando:
    docker-compose up -d --build
3 - Executar a seguinte URL no browser:
    http://localhost:8082/crud-pessoa-0.0.1-SNAPSHOT/cadastroPessoa.xhtml
